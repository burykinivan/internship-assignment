# 📁 Console File Manager (C++17)

Простой консольный файловый менеджер, написанный на C++17 с использованием стандартной библиотеки `<filesystem>`.

![Скриншот работы](screenshot.png)

## ✨ Возможности

- Просмотр содержимого директорий
- Навигация по папкам (вход / назад)
- Удаление файлов и папок (рекурсивно)
- Пометка одного элемента (Mark)
- Копирование помеченного элемента в текущую папку
- Перемещение (вырезание) помеченного элемента
- Просмотр текстовых файлов прямо в консоли

## 🖥️ Системные требования

| Компонент        | Минимальные требования                          |
|------------------|--------------------------------------------------|
| ОС               | Windows 7 / 8 / 10 / 11                          |
| Компилятор       | MSVC 2019 или новее (Visual Studio)              |
| Стандарт C++     | C++17                                            |
| Консоль          | Любая (cmd, PowerShell, Windows Terminal)       |
| ОЗУ              | 32 МБ                                            |
| Диск             | 5 МБ свободного места                           |


## 📥 Установка и запуск

### Способ 1: Скачать готовый .exe

1. Перейдите в раздел [Releases](https://github.com/ваш-логин/console-file-manager/releases)
2. Скачайте `manager.exe`
3. Запустите двойным щелчком

### Способ 2: Скомпилировать самостоятельно

#### С использованием Visual Studio (рекомендуется)

```bash
# Откройте "Developer Command Prompt for VS"
cl /EHsc /std:c++17 manager.cpp
# Запустите
manager.exe
```

#### Через Visual Studio IDE

1. Создайте новый **Console Application** проект на C++
2. Замените содержимое основного `.cpp` файла на код из `manager.cpp`
3. В настройках проекта установите **C++ Language Standard: ISO C++17**
4. Нажмите `Ctrl + F5` (Запуск без отладки)

#### Через MinGW (с осторожностью)

```bash
g++ -std=c++17 manager.cpp -o manager.exe -lwinmm
manager.exe
```

> ⚠️ MinGW не всегда корректно работает с `<conio.h>` – рекомендуется использовать MSVC.

## 🎮 Управление

| Клавиша           | Действие                                 |
|-------------------|------------------------------------------|
| `↑` / `↓`         | Перемещение курсора                       |
| `Enter`           | Войти в выбранную папку                   |
| `Backspace`       | Перейти на уровень вверх (родительская папка) |
| `D`               | Удалить выбранный файл / папку            |
| `M`               | Пометить текущий элемент                  |
| `C`               | Скопировать **помеченный** элемент в текущую папку |
| `X`               | Переместить (вырезать) **помеченный** элемент |
| `F`               | Просмотреть текстовый файл                |
| `Esc`             | Выход из программы                        |

## 📖 Примеры работы

### Копирование файла из одной папки в другую

```
1. Выберите файл → нажмите M (появится метка [MARK])
2. Перейдите в другую папку (Enter / Backspace)
3. Нажмите C → файл скопирован
```

### Просмотр текстового файла

```
1. Выберите любой .txt файл
2. Нажмите F → содержимое отобразится на экране
3. Нажмите любую клавишу для возврата
```

### Удаление папки

```
1. Выберите папку курсором
2. Нажмите D → папка будет удалена рекурсивно
```

## ⚠️ Известные ограничения

- Можно пометить **только один** элемент за раз
- Нет подтверждения при удалении (действие необратимо!)
- Не поддерживаются маски файлов (показ только *.txt и т.п.)
- Просмотр только текстовых файлов (бинарные отобразятся некорректно)
- Привязанность к Windows (не кроссплатформенный)
